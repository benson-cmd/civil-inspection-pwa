# 附件七與附件八 PDF 輸出規則

## 一、照片位置平面圖

- 每戶每樓層一頁。
- 上方顯示案件編號、鑑定日期、標的物地址。
- 中央顯示手繪平面圖。
- 圖上顯示紅色照片編號圓圈。
- 顯示紅色拍攝方向箭頭。
- 左下角圖例：
  - `○` 代表照片位置。
  - `→` 代表照片拍攝方向。
  - `X` 代表不便進入。

## 二、調查紀錄表

欄位：

照片編號、位置、全景、牆面、平頂、地坪、梁、柱、其他、現況、裂縫、裂縫寬(mm)、滲水、剝落、備註。

## 三、現況照片頁

- 每頁 2 張照片。
- 每張照片包含：
  - 照片編號。
  - 標的物地址。
  - 說明。
  - 照片。

## 說明自動組合規則

由 `component_type`、`condition_type`、`crack_width_mm` 與 `note` 組合。

範例：

- 牆面裂縫，裂縫寬 0.3mm。
- 平頂滲水。
- 全景現況。
