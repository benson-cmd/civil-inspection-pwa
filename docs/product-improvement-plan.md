# 現況鑑定報告系統完善計畫

## 參考來源

- 參考網站：`https://civilengineer.tw/app/`
- 參考重點：現場快速輸入、預設選項、照片說明自動生成、一鍵報告、測量資料記錄與成果檢核。
- 原則：參考功能方向，不複製文案、畫面或特定實作；本系統仍以「工程顧問報告工作台 + 公會格式報告輸出」為主軸。

## 產品定位調整

目前系統不應只定位為「附件七與附件八工具」，而應逐步升級為：

> 土木技師現況鑑定報告工作台：從案件建立、現場記錄、照片與測量資料、主文編輯到正式報告匯出的一體化 PWA。

## 參考網站帶來的功能啟發

### 一、現場輸入要更快

參考網站強調地址新增介面與數字鍵盤效率。本系統可增加：

- 標準地址欄位拆分：
  - 縣市。
  - 鄉鎮市區。
  - 路街。
  - 段。
  - 巷。
  - 弄。
  - 號。
  - 樓層。
- 仍保留完整地址欄位，可手動覆寫。
- 門牌、樓層、裂縫寬度等欄位使用 `inputMode="numeric"` 或 `inputMode="decimal"`。
- 標的物批次新增：
  - 同一路段多戶連續門牌。
  - 同一棟多樓層。
  - 可快速複製上一戶的用途、裝修與會勘狀況。

### 二、現況記錄要由「表單填寫」變成「快速點選」

目前附件七已經有位置與現況勾選，下一步可完善為：

- 預設選項群組：
  - 位置：全景、牆面、平頂、地坪、梁、柱、門窗、樓梯、陽台、外牆、其他。
  - 現況：現況、裂縫、滲水、剝落、白華、鏽蝕、破損、污漬、磁磚隆起、其他。
  - 方位：東側、西側、南側、北側、室內、室外、前側、後側。
- 使用者可新增自訂選項。
- 自訂選項應依使用者或公司帳號保存，之後案件可重複使用。
- 裂縫、尺寸、滲水範圍等需要數值時，動態顯示數字欄位。

### 三、照片說明要升級為語句生成器

目前 `caption.ts` 已有基本組合規則，下一步可升級為：

- 依中文語序組合：
  - 位置 + 現況。
  - 現況 + 尺寸。
  - 方位 + 構件 + 缺失。
- 範例：
  - `牆面裂縫，裂縫寬 0.3mm。`
  - `平頂滲水。`
  - `北側牆面油漆剝落。`
  - `地坪磁磚隆起。`
- 可手動編輯自動說明。
- 若使用者手動編輯，系統保留自訂說明，不再強制覆蓋。
- 可建立「說明預覽」區，讓現場人員完成一張照片前先確認文字。

### 四、報告匯出要形成一鍵流程

目前已有附件七/八匯出與正式報告規則，下一步應建立「完整報告匯出精靈」：

1. 檢查基本資料是否完整。
2. 檢查主文是否仍有預設 placeholder。
3. 檢查附件一至附件八是否齊備。
4. 檢查附件七是否有未上傳照片的點位。
5. 檢查附件七是否有無資料樓層，匯出時自動略過。
6. 產出：
   - 封面。
   - 目錄。
   - 主文。
   - 附件隔頁。
   - 附件一至附件八。
7. 匯出前顯示問題清單，讓使用者可回到對應頁面修正。

### 五、附件五與附件六應變成測量模組

參考網站包含傾斜測量記錄、計算與成果輸出。對本系統而言，附件五與附件六不應長期只是 PDF 上傳槽。

#### 附件五：水準測量

可規劃欄位：

- 測點編號。
- 測點位置。
- 初測高程。
- 複測高程。
- 高程差。
- 備註。
- 測點照片。
- 測點位置圖。

輸出：

- 水準測量成果表。
- 測點位置圖。
- 測點照片頁。

#### 附件六：傾斜率測量

可規劃欄位：

- 測線編號。
- 標的物地址。
- 測線位置。
- X 向 / Y 向。
- 上測點距離。
- 下測點距離。
- 樓高。
- 位移量。
- 傾斜率。
- 傾斜方向。
- 遠景照片。
- 近景照片。

計算：

- 由輸入數據自動計算位移量。
- 自動計算傾斜率。
- 超過門檻時提示。

輸出：

- 傾斜率測量成果表。
- 測線照片。
- 測點局部放大圖。
- 傾斜示意圖。

## 開發優先順序

### P0：先把資料保存做好

目標：讓現場資料不會因重新整理或換裝置消失。

- 案件 CRUD 接 Supabase。
- 使用者管理接 `ci_profiles`。
- 案件權限接 `ci_project_members`。
- 附件七資料接資料庫：
  - targets。
  - floors。
  - plan paths。
  - no-entry zones。
  - inspection points。
  - photos metadata。
- 附件七照片上傳 Supabase Storage。
- 附件八照片上傳 Supabase Storage。

### P1：提升現場輸入效率

- 地址拆分與快速建立多戶。
- 常用選項自訂。
- 數字欄位使用 iPad 友善鍵盤。
- 複製上一戶資料。
- 附件七照片點位與紀錄表操作再精簡。

### P2：照片說明語句生成器

- 擴充 `caption.ts`。
- 增加方位、材料、缺失程度。
- 支援自訂描述保留。
- 建立說明預覽與一鍵重建說明。

### P3：完整報告匯出

- 匯出前檢查清單。
- 正式報告封面。
- 目錄自動頁碼。
- 主文套公會頁首。
- 附件合併。
- 附件七/八不套主文頁首。

### P4：附件五與附件六測量模組

- 水準測量資料輸入。
- 傾斜率測量資料輸入。
- 自動計算與警示。
- 測點照片與局部圖。
- 測量成果 PDF。
